### `my_dict = {key1 : value1, key2 : value2, key3 : value3 }`

## 创建空字典
`emptyDict={}`或`emptyDict=dict()`
## 访问
```python
d = {'name':'wxm','age':'18',222:'a'}
print(d['name'])            #wxm
print(d[222])              #a
```

```python
My_Dict = {'list': [1,2,3],'dict' : {'a':4, 'b':5, 'c':6} ,'tup' : (1,2,3)}  
print(My_Dict['dict']['a'])   #4
```

## 修改

`d['age']=20`
与改数组一致

## 删除
##### 删除key`del My_Dict['tup']`
##### 删除字典`del My_Dict`
##### 清空字典`My_Dict.clear()`

## 方法
`len(My_Dict)`求字典内key 的个数
