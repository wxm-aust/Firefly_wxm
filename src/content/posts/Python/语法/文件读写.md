# 写文件
## open函数
##### `open('文件名','打开方式')`
|打开方式|效果|
|:--:|:--:|
|w|读写|
|r|只读|
|a|追加内容|

```python
text = '这个句子将被写进.txt'

my_file = open('my_file','w')
my_file.write(text)
my_file.close()
```
运行后桌面会生成一个`my_file.txt`内容是`这个句子将被写进.txt'`

```python
text = '\n这个句子将被写在上一句的后面'

my_file = open('my_file','a')
my_file.write(text)
my_file.close()
```
运行后`my_file`中变成
```
这个句子将被写进.txt
这个句子将被写在上一句的后面
```
# 读文件
```python
file = open('my_file','r')
content = file.read()
print(content)
```
运行结果
```
这个句子将被写进.txt
这个句子将被写在上一句的后面
```
##### 可用readline读取文件，按读取次数决定所读到的行数
```python
file = open('my_file','r')
content = file.readline()
print(content) #这个句子将被写进.txt
content = file.readline()
print(content) #这个句子将被写在上一句的后面
```
##### 可用readlines读取文件，文件每行内容将被存入列表的一格中
```python
file = open('my_file','r')
content = file.readlines()
print(content) 
```
运行结果
`['这个句子将被写进.txt','\n这个句子将被写在上一句的后面']`
