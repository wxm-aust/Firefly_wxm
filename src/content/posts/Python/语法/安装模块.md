# 部署镜像源
#### 对于windows
pycharm加入清华源解决一切问题

#### 对于linux
`sudo`调用管理员权限
`pip`或者`pip3`调用*安装模块*
`install`安装 *or* `uninstall`卸载
###### eg: 
`sudo pip install numpy`安装numpy
`sudo pip uninstall numpy`卸载numpy
`sudo pip install numpy-1.10.1`安装指定版本的numpy
`sudo pip install -U numpy`更新numpy

### 引入别名
上例每次使用都要加`turtle.`前缀，利用`as`可重定向以简写前缀

```python
import numpy  
a = numpy.array([[5,1,2,3],[4,5,6,7]])  
print(a[0][0])
```
等价于
```python 
import numpy as np  
a = np.array([[5,1,2,3],[4,5,6,7]])  
print(a[0][0])
```
