# print
`print()`为输出函数
* 通过`''`或`""`包住要打印的字符串，或直接写入变量名，中间用`,`隔开
* 如果要答应的字符串中有`'`，可通过转义字符调节冲突
* 结束一个`print`会自动换一行

|转义字符|效果|
|:--:|:--:|
| \ |消除符号歧义|
|\n|换行|
|\t|横向制表|
|v|纵向制表|
|a|响铃|
|b|退格|
|r|局部替换|
|f|换页|
|r|百分精度|
`print("\r{:3}%".format(i),end=' ')`
##### 方法
```python
name = 'alice'
print(name.title())
```
运行结果不是`alice`而是`Alice`，这是因为方法`title`作用于`name`

|方法|作用|
|:--:|:--:|
|title()|输出首字母大写|
|upper()|输出全部大写|
|lower()|输出全部小写|

### 列表实现
此处迭代器i是列表每一个元素而非它们的下标
```python
my_list = [1,2,3,5,4]
for i in my_list:
	print(i)
	
#运行结果
#12354
```
### range实现
#### `range(start,stop,step)`
* `range(n)`会生成一个从0到n-1的迭代器
故上一段代码等价于
```python
my_list = [1,2,3,5,4]
for i in range(len(my_list)):
	print(my_list[i])
```
* `range(l,r)`的范围实际上是l到r-1,即==l到r左闭右开== 
* 可以传入第三个参数控制步长
```python
for i in range (1,7,2):
	print(i)

#运行结果
#135
```

# 函数
* 统一用`def`来声明函数
* ==注意缩进==函数包括的所有内容都应该缩在def后面
* 函数传参不写数据类型
* def后定义参数时可以直接给参数赋值~~纯弱智功能，还有条条框框，不如在函数内部赋值~~
```python
def ppo(a):
	a=int(a)+1
	print(a)
```
在IDE中直接输入`ppo(1)`就可以调用函数，会打印`2`
也可以在程序中调用，在程序主体加入一行`ppo(a)`即可
可以用return传回函数的返回值,在主函数中打印
# 全局变量和局部变量
* 在def中的变量不能直接在主函数中使用
* 局部变量可以用`global`声明后变为全局变量
```python
def swap(x,y):
    global a
    a=y
    global b
    b=x
a=int(input())
b=int(input())
swap(a,b)
print(a,b)

```
