# 建立
### `list = ['a',123,'string]'`
# 访问
* 正向下标从0开始
* 反向下标从-1开始
```python
list = ['a',123,'string]
print(a) #a 123 string
print(a[0]) #a
print(a[-1]) #string
print(a[-3]) #a
```
* 可利用`:`截取列表片段
```python 
# 从第二位开始（包含）截取到倒数第二位（不包含）
print ("list[1:-2]: ", list[1:-2])
```
# 更新
## 添加
### `list[1] = 456`
也可以利用`append()`在列表末尾添加元素
	eg:`list.append('false')`
## 删除
### `del()`
`del(-1)`list会变成`['a',123]

# 操作符
* `len(list)`取列表长度
* `list1+list2`将两个列表拼接
* `[a]*4`后变成`[a,a,a,a]`
* `a in list` 判断a是否在list中，若在返回`True`
* `max(list)`返回list中的最大值
* `min(list)`返回list中的最小值
* `list.clear()`清空列表
* `list.count(x)`统计list中x的个数
* `list.insert(idx,x)`在idx处插入x

# 嵌套
```python
a = ['a', 'b', 'c']
n = [1, 2, 3]
x = [a, n]
x                 #[['a', 'b', 'c'], [1, 2, 3]]
x[0]              #['a', 'b', 'c']
```
