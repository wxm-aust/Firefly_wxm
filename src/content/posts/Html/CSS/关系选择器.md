##### 后代选择器
选择所有被E元素包含的F元素
语法`E F{}`
```css
ul li{
    color:green;
}
```
```html
<ul>
     <li>ul下的li是绿色</li>
     <li>ul下的li是绿色</li>
</ul>
 <ol>
     <li>ol下的li不是绿色</li>
</ol>
```

##### 子代选择器
选择所有作为E元素的**直接**子元素F，对更深一层的元素不起作用
语法`E>F{}`
```css
div>a{
    color:red
}
```
```html
<div> 
    <a>直接子元素变红</a>
    <p> <a href="#">孙元素不变色</a> </p>
    <a href="#">子元素变红</a>
</div>
```

##### 通用兄弟选择器
向下选择E元素后的第一个F元素
语法`E+F{}`
```css
h1+p{
    color:red;
}
```
```html
<h1>h1元素</h1> 
<p>紧跟的第一个元素变红</p> 
<p>第二个元素不变红</p>
```
##### 相邻兄弟选择器
向下选择E元素后的所有F元素
语法`E~F{}`
```css
h1~p{
    color:red;
}
```
```html
<h1>h1元素</h1>
<p>h1下第一个元素变红</p>
<p>h1下第二个元素变红</p>
```
