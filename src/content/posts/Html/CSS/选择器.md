### 全局选择器
`*`改变页面所有样式
* 能统一改变当前html的所有样式
* 常用于页面初始化
* 优先级很低
### 元素选择器
参考[[css]]中的代码
### 类选择器
`.`指定标签`class`，为设置样式提供便利
###### `class`
* 可以被多个标签共用
* 一个标签只能有一个类名
			 `<p class="name1" class="name2"`是错误的
* 类名不能以数字开头
* 同一个标签可以使用多个类选择器，用空格隔开
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<style>
    p{
        color: blue;
    }
    .reeeed{
        color: red;
    }
    .size{
        font-size: 30px;
    }
</style>
<body>
    <p class="reeeed size">
        这是一条红色的段落，并且为30px
    </p>
    <p class="no red">
        这不是一条红色的段落,它是蓝色的，class的优先级较高
    </p>
</body>
</html>
```

### ID选择器
`#`针对特定标签使用，只能使用一次 
###### `id`
* 一个id只对应一个标签，一个标签只有一个id
* id不能以数字开头
```css
	<style>
    #reeeed{
        color: red;
        font-size: 30px;
    }
</style>
```
```html
<body>
    <p id ="reeeed">
        这是一条红色的段落
    </p>
    <p id = "reeeed">
        这也是一条红色的段落，但如此操作不符合规范
    </p>
</body>
```

### 合并选择器
`选择器1,选择器2,...{}`可以提取共同样式以减少重复代码
```css
<style>
    h3,.reed,#reeeed{
        color: red;
        font-size: 30px;
    }
</style>
```
```html
<body>
    <p id ="reeeed">
        这是一条红色的段落
    </p>
    <p class = "reed">
        这也是一条红色的段落
    </p>
    <h3>这是一条红色的标题</h3>
</body>
```

## 优先级
内联样式>ID选择器>类选择器>元素选择器>全局选择器
