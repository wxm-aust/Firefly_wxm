##### `text-align`设置文本对齐方式
|值|效果|
|:---:|:---:|
|left|默认左对齐|
|right|右对齐|
|center|居中对齐|

##### `text-decoration`设置修饰

| 值           | 描述       |
| ------------ | ---------- |
| underline    | 定义下划线 |
| overline     | 定义上划线 |
| line-through | 定义删除线 |

##### `text-transform`设置文本自动大小写方案
| 值         | 描述                 |
| ---------- | -------------------- |
| captialize | 定义每个单词开头大写 |
| uppercase  | 定义全部大写字母     |
| lowercase  | 定义全部小写字母     |

##### `text-indent`设置文本自动缩进
* 单位为px
	`text-indent:50px;`
* 允许是负数，实现向左缩进
