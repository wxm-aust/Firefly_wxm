# 输出
`alert("要输出的内容");`在浏览器中弹出一个对话框,然后把要输出的内容展示出来
`document.write("要输出的内容"); `
在控制台输出内容
`console.log("要输出的内容");`

# 字符串
```js
var longString = 'Long\'long\'string';//反斜杠转义
longString //输出
longString.length//输出字符串长度
longString.charAt(1)//输出指定位置字符
					//类似char数组，下标从0开始，越界返回空
```
运行结果:
Long 'long' long string==待测试==o

`s1.concat(s2,s3)`拼接s1和s2,s3
	如果传入参数不是字符会自动转为字符

`s1.substring(begin,end)`或`s1.substring(begin,length)`
取出字串并返回
	下标从0开始
	不改变原字符串
	length留空默认总长度

`s1.indexOf('s2')`在s1中找s2，返回第一个匹配的位置（KMP)
`s1.trim()`去除s1两端的空格
`s1.split('x',num)`从s1的x处断开，返回一个成员数为num的数组
	分割关键字不在返回数组中出现
		传入空格就分割每一个字符
		省略参数就返回原字符串

# 数组
```js
var a=3
var arr = [ture, 2, a];
arr[3]='4';
```
可用多维数组
`arr.length`返回数组成员数量
遍历方法与c++完全相同
`arr.push(s1,s2)`在数组末端加入s1,s2
`arr.pop()`弹出数组最后一个元素并输出
`arr.shift()`删除数组最后一个元素并输出
`arr.unshift(s1)`在第一个元素前添加s1并返回新数组的成员数
`arr.join(s1)`输出整个数组并在输出结果中用s1分隔每个成员
`arr1.concat(arr2)`将arr1加入arr2前或将arr2接在arr1后
`arr.reverse`反转数组排列
`arr.indexOf(s1,location)`从location向后返回s1在arr中的位置
	位置即排名，从1开始
# 函数

```js

	console.log(x * x);
}

square(2) // 4
```

```js
function cube(x) {
	return (x * x  * x);
}
var ans = cube(3) 
console.log(ans); //9
```
