```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
</body>
</html>
```

`<!DOCTYPE html>`为网页声明，避免不同浏览器间的差异，写在最开始
`<html>`为双标签，表示定义html文档
	`<head>`为双标签，用于赋予属性
		`<title>`为双标签，用于改变网页标题
		`<meta>`为单标签，可以用来指定编码格式
	`<body>`为双标签，是网页的主体
