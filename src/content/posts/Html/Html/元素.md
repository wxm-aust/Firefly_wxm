# 块级元素
* 在页面中独占一行（自上而下）
* 可以设置width height 等属性
* 一般块级元素可以包含其他块级元素或者内联元素
常见的有`div` `form` `h1` `h6` `hr` `p` `table` `ul`
# 内联元素（行内元素）
* 不会独占一行，只占自身的大小
* 不能设置width height 等属性
* 可包含内联元素，不能包含行内元素
常见的有`a` `b` `em` `i` `span` `strong` 
## 行内块级元素
* 不会独占一行，只占自身的大小
*  可以设置width height 等属性
常见的有`button` `img` `input`
