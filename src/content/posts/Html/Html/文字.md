# 位置摆放
默认居左
在标签属性中添加`align = "center | right"`来改变
```html
<h1>居左</h1>
<h1 align = "center" > 居中 </h1>
```
在前标签里写的参数叫==标签属性==

# 水平线
`<hr/>`单标签，在页面中创建一条水平线
可以传入参数改变 颜色 宽度 粗细 位置 `hr color="" width="" size="" align="" />`
`/`为==结束标签的标志==一般写在">"前面
宽度和粗细单位为px
默认居中

|标签|描述|
|---|:---:|
|`<em>`|<em>着重文字</em>|
|`<b>`|<b>粗体文字</b>|
|`<i>`|<i>斜体文字</i>|
|`<strong>`|<strong>加重语气</strong>|
|`<del`|<del>删除文字</del>|
|`<span>`|<span>方便css的文字</span>|

# 超链接
`<a href = " url "> 链接文本 <a>`
url链接网页时需要带上协议，链接文件同样可用用相对路径和绝对路径
`url`填入`#`表示空链接
