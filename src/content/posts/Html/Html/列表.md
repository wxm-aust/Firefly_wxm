# 有序列表
`<ol>`开始一个有序列表
	在`<ol>`标签中可以加入标签属性`<type>`来决定以什么方式编号
`<li>`构造列表
### 例子
```html
	<ol type="A">
		<li>第一点内容</li>
			<ol>
				<li>第1.1条内容</li>
				<li>第1.2条内容</li>
			</ol>
		<li>第二点内容</li>
			<ol type="I">
				<li>第1.1条内容</li>
				<li>第1.2条内容</li>
			</ol>
		<li>第三点内容</li>	
	</ol>	
```
##### 运行结果
```
A. 第一点内容
	1.第1.1条内容
	2.第1.2条内容
B. 第二点内容
	I.第2.1条内容
	II.第2.2条内容
C. 第三点内容
```

|type=""|编号方式|
|---|:---:|
|1|1 2 3|
|a|a b c|
|A|A B C|
|i|i ii iii|
|I|I II III|

# 无序列表

`<ul>`开始一个无序列表
	在`<ol>`标签中可以加入标签属性`<type>`来决定小圆点的样子
`<li>`构造列表

|type=""|编号方式|
|---|:---:|
|disc|默认实心圆|
|circle|空心圆|
|square|小方块|
|none|不显示|
