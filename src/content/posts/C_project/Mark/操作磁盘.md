# 获取盘符
### `GetLogicalDrivers` 
```cpp
unsigned allDisk;
allDisk =  GetLogicalDrivers();
```

##### `GetLogicalDrivers()` 返回一个int类型数，转为二进制后与盘符一一对应

 eg:有C D E F 盘，则将倒数第3 4 5 6位变为1,如表
 
|   字节1    |   字节2    |   字节3    |   字节4    |
| :------: | :------: | :------: | :------: |
| 00000000 | 00000000 | 00000000 | 00111100 |
| ???????? | ???????? | ??????JI | HGFEDCBA |


则返回值为该二进制数`111100`转为十进制 `60`
##### 对返回值向右位移`>>`或者按位与`&`1判定结果`==`1即可判定盘符
```cpp
for(int i=0;i<26;i++){
	if(allDisk & 1 ==1) 
		printf("存在%c"盘",'A'+i);
	allDisk = allDisk >> 1;
}
```

# 判断磁盘类型
### `GetDriveType()`
```cpp
if(GetDriveType(u:) == DRIVE_REMOVEABLE)
	printf("这是一个 可移动磁盘")；
```

|DRIVE_|返回值|说明|
|---|:---:|:---:|
|DRIVE_|UNKNOWN|无法确定|
|DRIVE_|NO_ROOT_DIR|路径无效|
|DRIVE_|FIXED|硬盘或闪存|
|DRIVE_|REMOVEABLE|u盘|

# 获取文件属性
### `FindFirstFile`    `FindNextFile`
`WIN32_FIND_DATA`其本质为结构体，用于定义数据缓冲区
`FindFirstFile`获取指定目录下的第一个文件属性
`FindNextFile`获取first指定的文件的下一个文件的属性
```cpp
	WIN32_FIND_DATA findresult; //定义结构体保存结果
	HANDLE flag=FindFirstFile("U:\\*.*",&findresult); //*.*为全部文件 HANDLE创建flag为句柄
	printf("%s\n",finddata.xxx); //xxx为结构体内容 待选
	FindNextFile(flag,&findresult); //此时findresult存了第二个文件的属性
	printf("%s\n",finddata.xxx); 
```
* 通常可用FindNextFile判断是否有下一个文件，如果没有返回0
`if( FindNextFile(flag,&findresult) == 0 ) cout<<搜索结束<<endl;
