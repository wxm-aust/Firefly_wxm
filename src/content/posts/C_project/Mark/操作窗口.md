# 获取句柄
### `FindWindow`   `FindWindowEx`  `GetForegroundWindow`
`HWND`定义句柄
`FindWindow`获取顶层窗口(如Edge本体)，返回string类型句柄
`FindWindowEx` 获取子窗口(如Edge的每个标签页)，返回string类型句柄
`GetForegroundWindow`获取最前窗口，返回string类型句柄
每次开关程序都会分配新的句柄，故GFW及其获取的句柄在常用程序中并非永久可用
```cpp
HWND handle,flag;
handle = FindWindow("类名","标题");//获取指定窗口句柄
flag = GetForegroundWindow();   //获取当前打开的最前窗口句柄
while(handle != GetForegroundWindow()){  //判断最前窗口是否打开
	//寻找窗口的操作
	//参照本页[[操作窗口#^e2b817]]
} 
```

# 改变窗口位置
### `SetWindowPos`
```cpp
 SetWindowPos(handle,HWND_TOPMOST,0,0,0,0,xxx);
```
第一个参数传对象窗口句柄
第二个参数为置顶

|2.参数|效果|
|:--:|:--:|
|HWND_TOPMOST|窗口置顶|
|HWND_NOTOPMOST|解除窗口置顶|

第三、四个参数为窗口位置，以左上角为（0，0）
第五、六个参数为窗口大小
第七个参数为3.4.5.6参数的开关

|7.参数|效果|
|:--:|:--:|
|SWP_NOMOVE|使3.4参数无效|
|SWP_NOSIZE|使5.6参数无效|

# 获取窗口位置
### `GetWindowRect`
直接返回`bool`类型,判断窗口位置
==指针传入返回==可返回窗口位置,单位为px
```cpp
	RECT location;
	GetWindowRect(handle,&location); //将handle传入locattion
```

|缓存池|参数|返回值|备注|
|:--:|:--:|:--:|:--:|
|location|left|到左端距离|即X|
|location|right|到右端距离||
|location|top|到顶端距离|即Y|
|location|bottom|到底端距离||

# 操作光标
### `SetCursorPos(x,y)`  `mouse_event`
将光标移动到(x,y)   对鼠标按键进行操作
```cpp
	SetCursorPos(location.left + xxx , location.top + xxx);
	mouse_event(MOUSEEVENTF_LEFTDOWN,0,0,0,0);
	mouse_event(MOUSEEVENTF_LEFTUP,0,0,0,0);
```
