## 安装ros
```bash
wget http://fishros.com/install -O fishros && . fishros
```

## 安装Livox_SDK

```bash
git clone https://github.com/Livox-SDK/Livox-SDK2.git
cd ./Livox-SDK2/
mkdir build
cd build
cmake .. && make -j
sudo make install
```
## 安装Livox_ros_driver2

```bash
git clone https://github.com/Livox-SDK/livox_ros_driver2.git 
cd ~/ws_livox/src/livox_ros_driver2
./build.sh ROS1
source ~/ws_livox/devel/setup.sh   
```
## 其他依赖

```bash
sudo apt install ros-noetic-serial ros-noetic-tf2-sensor-msgs libceres-dev  libeigen3-dev ros-noetic-move-base-msgs ros-noetic-navigation
rosdep install -r --from-paths src --ignore-src --rosdistro noetic -y
```
